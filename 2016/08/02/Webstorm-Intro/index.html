什么是 WebStorm

WebStorm 是jetbrains公司旗下一款JavaScript 开发工具。被广大中国JS开发者誉为“Web前端开发神器”、“最强大的HTML5编辑器”、“最智能的JavaScript IDE”等。

WebStorm 的优点

  • JetBrains 公司对 WebStorm 功能的持续改进和维护

  • 几乎支持所有前端最新的语言

  • 自动保存，省去了ctrl + S的麻烦
  • 很多编辑器只要文件关闭了就没有历史记录了，但是 WebStorm 有，如果遇到突发情况（停电等），不至于丢失代码
  • 集成了很多功能，包括 Emmet 代码快速编写神器，Git、SVN 等版本控制系统，可以使用 Grunt、gulp 等构建工具
  • 主题多样，可根据个人喜好安装特定主题文件
  • 强力的代码分析系统，提示代码错误，防止后台程序猿吐槽
  • CSS 中颜色的智能显示，附带调色板
  • 插入图片是提示图片尺寸，插入以后也可以预览图片
  • 各种提高效率的快捷键：
    ctrl + D & ctrl + Y
    ctrl + W

WebStorm 的缺点

  • 占内存
    运存4G以上可忽略不计
  • 没有别的缺点了
