Chrome 插件集锦

Updated on: 2016.10.01 Published on: 2016.03.10 MelodyPro
  • Chrome

Chrome 大好前程

根据 Net Market Share 的报告，截止 2016年2月 Chrome 的市场份额已经达到了 36.56% 。这骄人的成绩，想必那些各种好玩有趣实用的 Chrome 插件也做了不少贡献吧。

入选名单（排名不分先后顺序）

WhatFont

每当我们要查看网页中的字体时，总是会习惯地右键 “检查” 或者 F12 进入开发者工具。实际上我们还有一种更优雅的方式来查看网页中的字体，那就是 WhatFont ：

咋一看还以为是什么数学公式 !_! 别慌，

这个插件使用起来很简单，只需要点击 Chrome 右上角的图标，就会出现 Exit WhatFont 按钮，表示已经开了启字体查看的功能。当鼠标在网页中移动时，会显示字体名称：

如果我们在点击一下鼠标，会出现一个“便签”，里面显示了包括 font-family、font-size、line-height、color 等信息。比 F12 优雅很多有没有。

Isometric Contributions

一个 GitHub 项目，出自帅哥 Jason Long 之手，可以瞬间让我们的 GitHub 提交次数统计变成 3D 的图表，视觉效果不错。安装好插件后，我们可以在以前和现在的图表进行切换，也可以同时显示两种图表。美中不足的就是没有动态交互效果。

图片来自 GitHub

印象笔记·剪藏

剪藏可以快速保存、标注和分享网页正文、图片、截屏或者网址。点击工具栏中的大象图标，打开剪藏。选择剪藏类型，我们可以保存网页正文、整个页面或选中部分。还能离线访问笔记，随时浏览或编辑已保存的内容。

Wappalyzer

Wappalyze 可以检测到一个网站使用的技术，包括内容管理系统、电子商务平台、web 服务器、JavaScript 框架、分析工具等等。详细清单可查看 官网。

Momentum

一个美化 chrome 新标签页的插件，每天一张精美的图片，外加 TODO、天气等小功能，开启崭新的一天。

FireShot

网页截图的好工具

划词翻译

支持谷歌、百度、有道、必应四大翻译和朗读引擎，可以方便的查看、复制和朗读不同引擎的翻译结果。
